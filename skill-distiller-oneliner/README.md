# Skill Distiller (One-Liner)

Quick reference variant of skill-distiller (~100 tokens).

Minimal TRIGGER/ACTION/RESULT format for when you just need a reminder.

## Installation

```bash
openclaw install leegitw/skill-distiller-oneliner
```

## Usage

```bash
/skill-distiller-oneliner path/to/skill.md
```
