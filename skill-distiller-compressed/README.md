# Skill Distiller (Compressed)

Self-compressed variant of skill-distiller (~975 tokens vs ~2,500).

Same functionality as [skill-distiller](../skill-distiller/), smaller context footprint.

## Installation

```bash
openclaw install leegitw/skill-distiller-compressed
```

## Usage

```bash
/skill-distiller-compressed path/to/skill.md
```
