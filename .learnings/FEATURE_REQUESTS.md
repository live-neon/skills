# Feature Requests

User-requested capabilities and enhancements.

Format: `[FEAT-YYYYMMDD-XXX]`

---

## Template

```markdown
### [FEAT-YYYYMMDD-XXX] Title

**Date**: YYYY-MM-DD
**Requester**: User / Agent / Review
**Priority**: High / Medium / Low

**Description**:
What capability is requested.

**Use case**:
Why it's needed.

**Status**: Pending / In progress / Completed / Declined

**See also**: Related observations
```
